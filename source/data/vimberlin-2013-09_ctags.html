# Installation

    # Linux
    sudo apt-get install exuberant-ctags
    # Mac
    brew install ctags


# Creating tags

    $ git clone https://github.com/padrino/padrino-framework
    $ cd padrino-framework
    $ ctags -R .


# Excluding certain files

    $ ctags -R --exclude="*.js" .

even exclude more than one file:

    $ ctags -R --exclude="*.js" --exclude="*.git".


# Understanding the tag format header


The header:

    !_TAG_FILE_FORMAT  2   /extended format; --format=1 will not append ;" to lines/
    !_TAG_FILE_SORTED  1   /0=unsorted, 1=sorted, 2=foldcase/
    !_TAG_PROGRAM_AUTHOR   Darren Hiebert  /dhiebert@users.sourceforge.net/
    !_TAG_PROGRAM_NAME Exuberant Ctags //
    !_TAG_PROGRAM_URL  http://ctags.sourceforge.net    /official site/
    !_TAG_PROGRAM_VERSION  5.9~svn20110310 //

# Tag navigation commands

- `<C-]>` ... jump to a file
- `<C-i/C-o>` ... move forward/back
- `:tag <tagname>` ... search after the tagname
- `:tselect` ... select the tag you want if matches are unambiguous
- `:tnext, :tprevious, :tfirst, :tlast` ... left for you as an exercise


# Comfort

- use Autotags.vim http://www.vim.org/scripts/script.php?script_id=1343 for tag update
